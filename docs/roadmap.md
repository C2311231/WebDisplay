# Planned Development Roadmap

## Development Milestones

1. v2.0.0-dev.1 Planning Completed
    - Planning documentation completed for the continuation of this project.
    - Repository cleaned up of systems that will be removed or no longer relevent.

2. v2.0.0-dev.2 Device Registration
    - Completion of basic device registration.
    - Complete basic device control api.

3. v2.0.0-dev.3 Displaying Simple Content
    - Implement client device website content displaying functionality.
    - Implement schedule following automation for content.

4. v2.0.0-dev.4 CEC Control
    - Implement CEC display controls and configuration.
    - Automate controls with content display.

5. v2.0.0-dev.5 Configuration Syncing
    - Connect client and server devices over api.
    - Automatically sync configurations and schedules from the server.

6. v2.0.0-dev.6 Server API
    - Implement and document server control api.

## Pre-release Milestones

1. v2.0.0-alpha.1 Web UI
    - Create a web interface for the server for controling and managing devices.

2. v2.0.0-alpha.2 Content Managment
    - Add ability to create multiple pieces of content to be scheduled independently.

3. v2.0.0-alpha.3 Multidevice Managment
    - Formalize control over multiple devices.
    - Add grouping ability and group scheduling.

4. v2.0.0-alpha.4 Feedback
    - Add in device status monitoring and real time controls.

5. v2.0.0-alpha.5 Cacheing
    - Add device content caching and offline operation.

6. v2.0.0-alpha.6 Advanced Content
    - Add additional content options including, video, presentions, slideshows, and images.

7. v2.0.0-alpha.7 Onboarding
    - Add in easy pairing abilities and authentication.
